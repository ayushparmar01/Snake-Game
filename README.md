# Snake Game using Python

## Description

This is a simple Snake Game built using Python and Pygame. In this game, the player controls the snake using arrow keys and tries to eat the food to increase the score. The game ends when the snake touches the boundary.

I created this project to improve my Python programming skills and understand basic game development using Pygame.



## Features

* Classic snake game logic
* Keyboard controls using arrow keys
* Score display
* Random food generation
* Game over screen
* Option to play again or quit



## Tech Used

* Python
* Pygame
* Random module
* Time module


## How to Run

1. Install Python on your system

2. Install Pygame
   pip install pygame

3. Clone this repository
   git clone https://github.com/your-username/snake-game.git

4. Go to the project folder
   cd snake-game

5. Run the file
   python snake.py



## Controls

* Left Arrow → Move Left
* Right Arrow → Move Right
* Up Arrow → Move Up
* Down Arrow → Move Down
* P → Play Again
* Q → Quit Game
